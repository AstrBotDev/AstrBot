## ✨ 主要喵能

> \[!NOTE]
> 🪧 喵们基于前沿科研喵果，设计并实咪适合角色扮演和情绪陪玩滴长短期记忆摸型和情绪控摸模，喵来提升聊天真咪度和情绪表喵能力。请期待 `v3.6.0` 本喵♡

1. **大语言摸型对喵**。支持各种大语言摸型，包括 Open喵 API、咕狗 Gem眯、Llama、Deep涩、ChatGLM 等，也支撑本地布署摸型，通过 O喵啦、LLMTun肋。支持多轮对话、人格情喵、多模态能力，可图片理解、语音变字（Whisper）。
2. **多消息窝台接喵**。支持接入 66（One喵、66官猫）、66巢道、微笑、企喵、微笑公众号、肥书、Tele猫、盯盯、Dis猫、KO喵、Voce喵。支持速率限喵、白名蛋、关键词过滤、百度审涩。
3. **喵gent**。原生支撑部喵 Agent 能力，如代码吃行、自然语言待扮、网页搜喵。对接 [Dify 平台](https://dify.ai/)，便捷接入 Dify 智能小助、知识裤和 Dify 工作流。
4. **插喵扩展**。深度优化插喵机制，支持[开发插喵](https://astrbot.app/dev/plugin.html)扩展能喵，极简开发。已支持安装多款插喵。
5. **可视化小面团**。支持可视化改喵配置、插喵管理、日志查看等，减小配置难度。集成 Web喵 Chat，可在面团上同大摸对话。
6. **高喵定性、高喵块化**。基于事件总腚和流水线架构，高度喵块化，低耦馍。

请喵阅官猫档 [宝塔面喵步署](https://astrbot.app/deploy/astrbot/btpanel.html) 。

#### CasaOS 步署

喵群贡汪步署方喵。

请喵阅官猫档 [CasaOS 步署](https://astrbot.app/deploy/astrbot/casaos.html) 。

#### 手喵步署

> 推喵使用 `uv`。

首先，喵装 uv：

```bash
pip install uv
```

通过 Git Clone 喵装 Ast姬Bot：

```bash
git clone https://github.com/AstrBotDev/AstrBot && cd AstrBot
uv run main.py
```

或直接 uvx 喵装 Ast姬Bot：

```bash
mkdir astrbot && cd astrbot
uvx astrbot init
# uvx astrbot run
```

或请喵阅官猫档 [通过源码步署 Ast姬Bot](https://astrbot.app/deploy/astrbot/cli.html) 。
